# 剧情工具项目状态

## 产品定位

面向游戏和互动剧情设计的线性剧情编排工具。当前版本不做剧情分支和多结局，重点支持剧情内容、地图路线、情绪曲线和实时预览。

## 已完成里程碑

- M1 静态界面原型
- M2 剧情内容编辑 MVP
- M3 地图板块 MVP
- M4 情绪曲线 MVP
- M5 本地保存与导入导出
- M6 体验打磨

## 当前能力

- 新建、编辑、删除剧情场景
- 复制场景到内部剪贴板，并通过粘贴创建副本
- 场景拖拽排序
- 玩家视角和设计师视角实时预览
- 地图路线自动由地点生成，按剧情顺序从左到右显示
- 情绪曲线根据场景顺序和情绪强度实时生成
- 情绪类型自动带出默认强度
- 本地自动保存
- 导入 JSON
- 导出 JSON
- 导出 Markdown
- 保存时校验必填字段并标红

## 当前导入格式

仅支持 JSON。Markdown 当前只作为阅读和归档导出格式，不支持导入。
